--------------------------------------------------------------------------------
General:
--------

pynmd is a collection of tools that we have developed for analyzing and
processing oceanographic data.

Dependencies:
-------------
scipy, numpy, netCDF4, matplotlib, pandas

--------------------------------------------------------------------------------
Installing:
-----------
Set your PYTHONPATH environment variable. 
  In CSH
    setenv PYTHONPATH "/home/shusin2/shared/nmg"

Cloning:
--------
So far the primary version is located under
/home/shusin2/shared/nmg

To get a local copy of the project clone using ssh:
>> cd to local working directory
>> git clone ssh://user@server.oce.orst.edu/home/shusin2/shared/nmg/pynmd


Git Conventions:
----------------
1. Do not commit .pyc or .py~ files. To ignore these files automatically, go to 
   the top directory and execute
   >>  git config --global core.excludesfile .gitignore
